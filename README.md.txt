# Satellite Imagery Based Property Valuation

This project builds a multimodal regression pipeline combining tabular housing data with satellite imagery to predict property prices.

## Project Structure

- data/  
  - train.csv  
  - test.csv  

- data_fetcher.py – Fetches satellite images using latitude & longitude  
- preprocessing.ipynb – EDA and preprocessing  
- model_training.ipynb – Model training and evaluation  
- enrollno_final.csv – Final predictions  
- report/ – Final PDF report  

## Status
Work in progress. Final submission will include trained models, predictions, and report.
